# Codex CLI 安装指南

## 1. 安装

使用 npm 安装 Codex CLI。

```bash
npm i -g @openai/codex
```

## 2. 运行

在终端中运行 Codex。它可以检查你的代码库、编辑文件并运行命令。

```bash
codex
```

首次运行 Codex 时，系统会提示你登录。使用你的 ChatGPT 账户或 API 密钥进行身份验证。

如果不确定哪些计划包含 Codex 访问权限，请查看[定价页面](https://openai.com/pricing)。

## 3. 升级

Codex CLI 会定期发布新版本。查看[更新日志](https://github.com/openai/codex-cli/releases)了解发布说明。要使用 npm 升级，请运行：

```bash
npm i -g @openai/codex@latest
```
